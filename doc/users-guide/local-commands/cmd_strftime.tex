\section{\_STRFTIME}
\label{cmd:_STRFTIME}

\paragraph{Syntax:}
\subparagraph{}
\texttt{\_STRFTIME <time> "<format-string>" <variable> [Local|GMT]}

\paragraph{Purpose:}
\subparagraph{}
Stores the given time (in milliseconds) as a string in $<$variable$>$. 
The string is formatted as defined in $<$format-string$>$.
