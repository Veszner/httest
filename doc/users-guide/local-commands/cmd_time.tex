\section{\_TIME}
\label{cmd:_TIME}

\paragraph{Syntax:}
\subparagraph{}
\texttt{\_TIME <variable>}

\paragraph{Purpose:}
\subparagraph{}
Stores the current time in milliseconds in $<$variable$>$. The time that is returned represents the number of seconds elapsed since 00:00 hours, Jan 1, 1970 UTC. Itâ€™s also called UNIX EPOCH time.
