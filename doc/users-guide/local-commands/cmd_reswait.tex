\section{\_RESWAIT}
\label{cmd:_RESWAIT}

\paragraph{Syntax:}
\subparagraph{}
\texttt{\_RESWAIT}

\paragraph{Purpose:}
\subparagraph{}
Combines the \texttt{\_RES} and the \texttt{\_WAIT} command, 
ignoring connections that are not sending any data at all.
