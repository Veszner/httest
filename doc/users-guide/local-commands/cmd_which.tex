\section{\_WHICH}
\label{cmd:_WHICH}

\paragraph{Syntax:}
\subparagraph{}
\texttt{\_WHICH <variable>}

\paragraph{Purpose:}
\subparagraph{}
Stores the concurrency number of the current thread 
in $<$variable$>$.
