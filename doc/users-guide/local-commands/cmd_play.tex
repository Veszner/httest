\section{\_PLAY}
\label{cmd:_PLAY}

\paragraph{Syntax:}
\subparagraph{}
\texttt{\_PLAY SOCKET | VAR <variable>}

\paragraph{Purpose:}
\subparagraph{}
Plays back recorded data, either on the current socket 
or into a variable.
