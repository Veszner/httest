\section{\_CHUNK}
\label{cmd:_CHUNK}

\paragraph{Syntax:}
\subparagraph{}
\texttt{\_CHUNK}

\paragraph{Purpose:}
\subparagraph{}
This marks the end of a chunk data block. It counts the 
number of bytes sent since the last invocation of \texttt{\_FLUSH} 
and automatically adds chunk info to the request.
