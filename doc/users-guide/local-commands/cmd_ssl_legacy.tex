\section{\_SSL\_LEGACY}
\label{cmd:_SSL_LEGACY}

\paragraph{Syntax:}
\subparagraph{}
\texttt{\_SSL\_LEGACY on|off}

\paragraph{Purpose:}
\subparagraph{}
Turns SSL legacy behaviour on or off for renegotiation (for 
OpenSSL libraries 0.9.81 and above).
