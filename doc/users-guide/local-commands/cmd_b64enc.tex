\section{\_B64ENC}
\label{cmd:_B64ENC}

\paragraph{Syntax:}
\subparagraph{}
\texttt{\_B64ENC "<string>" <variable>}

\paragraph{Purpose:}
\subparagraph{}
Stores a Base64-encoded string into $<$variable$>$.
