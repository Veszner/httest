\section{\_SSL\_BUF\_2\_CERT}
\label{cmd:_SSL_BUF_2_CERT}

\paragraph{Syntax:}
\subparagraph{}
\texttt{\_SSL\_BUF\_2\_CERT pem cert}

\paragraph{Purpose:}
\subparagraph{}
Reads the given buf as a PEM certificate, usable with 
\texttt{\_SSL\_CERT\_VAL}.
