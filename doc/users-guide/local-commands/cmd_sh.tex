\section{\_SH}
\label{cmd:_SH}

\paragraph{Syntax:}
\subparagraph{}
\texttt{\_SH shell script line or END}

\paragraph{Purpose:}
\subparagraph{}
Embeds a shell script within a tmp file, and executes
it when \texttt{END} is found.
