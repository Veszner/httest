\section{\_RES}
\label{cmd:_RES}

\paragraph{Syntax:}
\subparagraph{}
\texttt{\_RES}

\paragraph{Purpose:}
\subparagraph{}
Waits for a connection accept. This is used within 
a \texttt{SERVER} body to receive incoming requests 
from clients.
