\section{\_ONLY\_PRINTABLE}
\label{cmd:_ONLY_PRINTABLE}

\paragraph{Syntax:}
\subparagraph{}
\texttt{\_ONLY\_PRINTABLE on|off}

\paragraph{Purpose:}
\subparagraph{}
Replaces all characters of ASCII code lower than 32 and 
higher than 127 with a blank (space) character.
