\section{\_LOG\_LEVEL}
\label{cmd:_LOG_LEVEL}

\paragraph{Syntax:}
\subparagraph{}
\texttt{\_LOG\_LEVEL <level>}

\paragraph{Purpose:}
\subparagraph{}
Sets the log level. $<$level$>$ must be a number 
from 0 - 4.
