\section{\_IGNORE\_BODY}
\label{cmd:_IGNORE_BODY}

\paragraph{Syntax:}
\subparagraph{}
\texttt{\_IGNORE\_BODY on|off}

\paragraph{Purpose:}
\subparagraph{}
Recieves the request body data, but does not store it anywhere 
or inspect it. This is used for performance testing.
