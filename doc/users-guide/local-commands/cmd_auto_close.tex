\section{\_AUTO\_CLOSE}
\label{cmd:_AUTO_CLOSE}

\paragraph{Syntax:}
\subparagraph{}
\texttt{\_AUTO\_CLOSE on|off}

\paragraph{Purpose:}
\subparagraph{}
"\texttt{\_AUTO\_CLOSE on}" lets \texttt{httest} automatically 
close a connection when an incoming "\texttt{Connection: close}" 
header is received.
