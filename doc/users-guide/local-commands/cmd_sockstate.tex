\section{\_SOCKSTATE}
\label{cmd:_SOCKSTATE}

\paragraph{Syntax:}
\subparagraph{}
\texttt{\_SOCKSTATE <variable>}

\paragraph{Purpose:}
\subparagraph{}
Stores connection state \texttt{CLOSED} or \texttt{CONNECTED} 
of the current connection in the variable.
