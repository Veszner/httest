\section{\_SYNC}
\label{cmd:_SYNC}

\paragraph{Syntax:}
\subparagraph{}
\texttt{\_SYNC}

\paragraph{Purpose:}
\subparagraph{}
Waits until the next full second is reached. This can be 
used to synchronize code that is executed in separate 
threads, such as multiple \texttt{CLIENTs} or a \texttt{CLIENT} 
and a \texttt{SERVER}.
