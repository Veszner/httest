\section{\_CLOSE}
\label{cmd:_CLOSE}

\paragraph{Syntax:}
\subparagraph{}
\texttt{\_CLOSE}

\paragraph{Purpose:}
\subparagraph{}
Closes the current connection (and sets the connection 
state to \texttt{CLOSED}).
