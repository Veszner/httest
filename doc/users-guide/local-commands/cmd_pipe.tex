\section{\_PIPE}
\label{cmd:_PIPE}

\paragraph{Syntax:}
\subparagraph{}
\texttt{\_PIPE [chunked [<chunk\_size>]]}

\paragraph{Purpose:}
\subparagraph{}
Starts a pipe for streaming the output of \texttt{\_EXEC} 
into the socket output stream, with optional chunk support.
