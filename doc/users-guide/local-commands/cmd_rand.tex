\section{\_RAND}
\label{cmd:_RAND}

\paragraph{Syntax:}
\subparagraph{}
\texttt{\_RAND <start> <end> <variable>}

\paragraph{Purpose:}
\subparagraph{}
Generates a random number between $<$start$>$ and $<$end$>$ 
and stores the result in $<$variable$>$.
