\section{\_FLUSH}
\label{cmd:_FLUSH}

\paragraph{Syntax:}
\subparagraph{}
\texttt{\_FLUSH}

\paragraph{Purpose:}
\subparagraph{}
Flushes the cached lines of data that must be sent in a 
request.

\subparagraph{}
NOTE: The calculation of the content length value for an 
\texttt{AUTO} content-length header is performed by this command!
