\section{\_SLEEP}
\label{cmd:_SLEEP}

\paragraph{Syntax:}
\subparagraph{}
\texttt{\_SLEEP <milliseconds>}

\paragraph{Purpose:}
\subparagraph{}
Stops execution for the defined amount of time (number 
of milliseconds).
