\section{GLOBAL}
\label{cmd:GLOBAL}

\paragraph{Syntax:}
\subparagraph{}
\texttt{GLOBAL <variable 1> [<variable 2> .. <variable N>]}

\paragraph{Purpose:}
\subparagraph{}
Defines the given variables as global, which are shared over all
\texttt{CLIENTs}, \texttt{SERVERs} and \texttt{DAEMONs}.
